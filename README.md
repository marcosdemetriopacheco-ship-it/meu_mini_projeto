
# Mini Projeto em HTML e JavaScript

Este é um projeto simples criado para aprender a diferença entre:

- index.html
- app.js
- README.md

## Como usar

1. Faça o download do projeto
2. Abra o arquivo `index.html` no navegador
3. Clique no botão exibido na tela

## Tecnologias usadas

- HTML
- JavaScript puro

## Objetivo

Aprender a estrutura básica de um projeto web.
